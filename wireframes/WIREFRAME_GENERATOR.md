## Navigation Arrow Pattern

Each wireframe should include:
- Green arrow (→) on right side for "Next" action
- Yellow arrow (←) on left side for "Back" action
- Flow indicator showing current step
- Clickable elements that navigate to related pages

## Implementation

All wireframes use:
- Dark theme (#111 background, #1a1a1a frame)
- Mobile frame (375x812px)
- Tailwind CSS via CDN
- Navigation arrows with pulse animation
- Flow indicators
